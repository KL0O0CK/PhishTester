# PhishTester

Интерактивный тренажер по распознаванию фишинга в реальном времени.

## Что делает проект

- Показывает два окна: безопасный сайт и фишинговую копию.
- Пользователь ищет 3 уязвимости на поддельной странице.
- При клике отображается подсказка с правовым контекстом РБ.
- После нахождения всех признаков показывается итоговое сообщение.

## Технологии

- HTML
- CSS
- Чистый JavaScript

## Как запустить

1. Открой файл `index.html` в браузере.
2. Нажимай на подозрительные элементы в правом окне.
3. Найди все 3 признака фишинга.

## Применение в заявке
